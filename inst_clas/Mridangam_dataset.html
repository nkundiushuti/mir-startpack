The Mridangam Stroke Dataset¶

The MTG proposes a few datasets that may be used for the instrument classification task:

  • IRMAS - containing annotated samples of multiple instruments

  • Good Sounds - containing monophonic sounds (notes and scales) for various instruments

  • Mridangam Stroke containing isolated drum strokes and the annotated tonic

All these datasets may be downloaded from zenodo. In addition, these datasets may be loaded and validated with easy using the mirdata library.

For educational purposes, we will work with the Mridangam Stroke dataset.

Instalation of packages¶

To download, validate, and load the data we use the mirdata library. We use MTG’s essentia for audio loading and feature computation. Matplotlib’s pyplot is used for plotting and pandas for data stats. We install these libraries through PyPI.
